# Landing Page

This is a project from The Odin Project Foundations course.

## Project Description

The goal of this project is to recreate a landing page layout using HTML and CSS.
It focuses on practicing page structure, styling, and layout techniques.

## Skills Demonstrated

* Structuring a webpage with HTML
* Styling elements with CSS
* Using Flexbox for layout
* Working with containers and sections
* Aligning content and spacing elements
* Using Git and GitHub for version control

## What I Learned

This project helped me understand how to structure and style a complete webpage.
I practiced using CSS to control layout, spacing, and alignment, especially with Flexbox.
